# AVAX Status Bar

An [Atom](https://atom.io/) package to keep an eye on the latest [AVAX](https://www.avalabs.org/) price while you code!

The price is provided by [CryptoCompare](https://www.cryptocompare.com/)'s API, expressed in USD.
